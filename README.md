# SQL HR Database Project (Beginner to Advanced)

This repository contains a complete **SQL practice project based on an HR Database**, designed to help **students and beginners learn SQL from basic to advanced level**.
The project covers real-world SQL use cases using employee, department, customer, order, and product data.  
Each assignment focuses on different SQL concepts, with **SQL query files and solution screenshots** provided.

## Purpose of This Project
- Help students understand SQL step-by-step
- Practice real HR and sales database queries
- Learn how to write clean, efficient, and analytical SQL
- Prepare for SQL interviews and data analyst roles

---

##  What This Project Covers
- Basic SQL queries (SELECT, WHERE, ORDER BY)
- Joins (INNER, LEFT, RIGHT, FULL)
- Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
- GROUP BY and HAVING
- Subqueries and correlated subqueries
- Common Table Expressions (CTEs)
- CASE statements
- String and date functions
- Real-world HR and sales data analysis
